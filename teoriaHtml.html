Curso HTML

HTML
es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto”, HTML describe la estructura de una página web, HTML consiste en una serie de elementos, Los elementos HTML le dicen al navegador cómo mostrar el contenido, Los elementos HTML están representados por etiquetas tambien llamados elementos.
Las etiquetas HTML etiquetan piezas de contenido como "encabezado", "párrafo", "tabla", etc.

Concepto DOCTYPE

!DOCTYPE html declaración define este documento como HTML5

Estructura HTML

La estructura de Html esta compuesta por una serie de etiquetas las cuales son las principales en donde iran las bases de nuestro documento.
Lo principal es especicar que el documento es HTML con la etiqueta "!DOCTYPE html" con "<" menor que para abrir y ">" mayor que para cerrar, "<>" Se utiliza para abrir y cerras toda etiqueta, cada etiqueta tiene una etiqueta de cierre con un "/" y el nombre de la etiqueta.
Luego la etiqueta "html" que es la raiz del documento.
Seguida de etiqueta "head" donde va la metainformacion sobre el documento
Seguida de la etiqueta "body" donde va el contenido visible de la pagina
Ejemplo:
!DOCTYPE html
html>
head>
title>titulo de la pagina /title>
/head>
body>
h1>My primer titulo./h1>
p>My primer parrafo./p>
/body>
/html>
Nota: Elimine los < para que se pudiera mostrar en la pagina.

Etiqueta Html

El elemento html define todo el documento. Tiene una etiqueta de inicio y una etiqueta de finalización

Etiqueta Head

El elemento head es un contenedor de metadatos. Los metadatos HTML son datos sobre el documento HTML. Los metadatos no se muestran. El elemento head se coloca entre la etiqueta html y la etiqueta body Los metadatos no se muestran. Los metadatos generalmente definen el título del documento, el juego de caracteres, los estilos, los scripts y otra metainformación.

Etiqueta Meta

El elemento "meta" se utiliza para especificar qué conjunto de caracteres se utiliza, descripción de la página, palabras clave, autor y otros metadatos. Los navegadores (cómo mostrar contenido), los motores de búsqueda (palabras clave) y otros servicios web utilizan los metadatos. ejemplo :
meta charset="UTF-8">
meta name="viewport" content="width=device-width, initial-scale=1.0">
meta name="description" content="Free Web tutorials">
meta name="keywords" content="HTML, CSS, XML, JavaScript">
meta name="author" content="Dainer villarreal">
Nota: Elimine los < para que se pudiera mostrar en la pagina.

Etiqueta Title

El elemento title define el título del documento y es obligatorio en todos los documentos HTML. El elemento title define un título en la pestaña del navegador proporciona un título para la página cuando se agrega a favoritos, muestra un título para la página en los resultados del motor de búsqueda.

Etiqueta Body

El elemento body define el cuerpo del documento Tiene una etiqueta de inicio y una etiqueta de finalización.

Etiquetas en Html

Las etiquetas HTML son nombres de elementos rodeados de corchetes angulares: Las etiquetas HTML normalmente vienen en pares como p> y /p> La primera etiqueta en un par es la etiqueta de inicio, la segunda etiqueta es la etiqueta final La etiqueta final se escribe como la etiqueta inicial, pero con una barra diagonal insertada antes del nombre de la etiqueta

Atributo Id

El atributo id especifica una identificación única para un elemento HTML (el valor debe ser único dentro del documento HTML). ejemplo :

Nota: El valor de identificación distingue entre mayúsculas y minúsculas. El valor de identificación debe contener al menos un carácter y no debe contener espacios en blanco (espacios, tabulaciones, etc.).
Diferencia entre clase e identificación : Un elemento HTML solo puede tener una identificación única que pertenezca a ese único elemento, mientras que varios elementos pueden usar un nombre de clase

Formulario Get

El método predeterminado al enviar datos del formulario es GET. Sin embargo, cuando se utiliza GET, los datos del formulario enviado serán visibles en el campo de dirección de la página ejemplo:
form action="/action_page.php" method="get"
....
/form

Nota : los datos del formulario enviado serán visibles en el campo de dirección de la página ejemplo:
/action_page.php?firstname=Mickey&lastname=Mouse

Formulario Post

si los datos del formulario contienen información confidencial o personal. El método POST no muestra los datos del formulario enviado en el campo de dirección de página.

form action="/action_page.php" method="post"

/form>

Rutas de archivos

Una ruta de archivo describe la ubicación de un archivo en la estructura de carpetas de un sitio web. Las rutas de archivo se utilizan al vincular a archivos externos como: páginas web, Imágenes, Hojas de estilo, JavaScripts
img src="picture.jpg"> picture.jpg se encuentra en la misma carpeta que la página actual
img src="images/picture.jpg"> picture.jpg se encuentra en la carpeta de imágenes en la carpeta actual
img src="/images/picture.jpg"> picture.jpg se encuentra en la carpeta de imágenes en la raíz de la web actual
img src="../picture.jpg"> picture.jpg se encuentra en la carpeta un nivel más arriba de la carpeta actual

Rutas de archivo absoluta

Una ruta de archivo absoluta es la URL completa de un archivo de Internet ejemplo:
img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">

Una ruta de archivo relativa apunta a un archivo relativo a la página actual. En este ejemplo, la ruta del archivo apunta a un archivo en la carpeta de imágenes ubicada en la raíz de la web actual ejemplo:

etiquetaDel

El elemento del HTML define el texto eliminado / eliminado ejemplo:

El elemento ins HTML define el texto insertado / agregado ejemplo:

Etiqueta Base

El base elemento especifica la URL base y el objetivo base para todas las URL relativas en una página ejemplo:
base href="https://www.w3schools.com/images/" target="_blank">
